# Trademark Pre-Screen — «Miss Lana's Fairy-Tale Theater»

**Дата:** 2026-06-22 · **Тип:** предварительный конфликт-скан для передачи trademark-юристу.

> ⚠️ **Это НЕ юридическое заключение.** Предварительная проверка по открытым источникам. Формальную чистоту знака и право на регистрацию определяет trademark-юрист (knockout-поиск перед подачей).

## 1. Объект проверки

- **Бренд (флагман):** **Miss Lana's Fairy-Tale Theater**
- **Зонтик/персона:** **Miss Lana** (под ней — школьные шоу, дни рождения, костюмированные персонажи)
- **Сфера → класс:** живой костюмированный детский театр + выездные шоу/праздники → ориентир **USPTO Class 41** (entertainment / theatrical / educational services; «live theatrical performances for children»)
- **Рынок:** США, англоязычный (база — Лос-Анджелес)
- **Активы (подтверждено владельцем):** домен **misslanatheater.com — свободен**; хэндл **@misslanatheater (Instagram) — свободен**

## 3. Предварительный вывод

Явного блокера не найдено; **риск иска — низкий**. Защитимое ядро — **«Miss Lana's» + логотип**; описатель «Fairy-Tale Theater» родовой (низкий риск нарушения, но не «занимается»). Единственная заметная вещь «на глаз юристу» — близость к **Faerie Tale Theatre** Шелли Дюваль (иное написание/носитель → низкий–умеренный).

---

## 4. Юристу — формально проверить

1. **Knockout-поиск по Class 41** (live theatrical / entertainment / educational services для детей): нет ли сходных «Lana / Miss Lana / Miss Lana's» (live + pending).
2. Оценить риск против **Faerie Tale Theatre** (Duvall) и любых действующих регистраций/прав, связанных с ним.
3. Проверить бренд одежды **Miss Lana (Cl. 25)** на предмет расширений в Cl. 41.
4. Совет по подаче: что регистрировать — вероятно **«Miss Lana's» (словесный знак) + логотип**; нужен ли дисклеймер на описательное «Fairy-Tale Theater».
5. Полный **common-law** поиск (не только реестр USPTO).

---

## 5. Снизить риск (в нашей власти)

- Написание — американское: **Fairy-Tale Theater** (не «Faerie Tale Theatre»).
- Не подражать оформлению/стилистике сериала Дюваль.
- Везде использовать **полное** «Miss Lana's Fairy-Tale Theater», не голое «Fairy Tale Theater».
- Застолбить активы **до публичного запуска:** misslanatheater.com + @misslanatheater (+ FB-страница; по возможности тот же хэндл на прочих площадках).
- Не вкладываться в логотип/брендинг и не делать публичный запуск, пока юрист не дал добро.
