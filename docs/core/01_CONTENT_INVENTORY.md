## Формат продукта (база для всех страниц)

**Театрализованное шоу:** костюмированный спектакль по сказке (**~30 мин**) **+
интерактивный перформанс с детьми** (**игры, танцы, мыльные пузыри**). Живые актёры.
Всего **около часа** (~30 мин спектакль + ~30 мин интерактив), дети **2–10 лет**.
Сквозная тема всего репертуара — **добро, дружба, взаимопомощь**. *(Длительность —
приблизительная. Формат «куклы внутри» подтверждён владельцем — guardrails переписать
отдельным пассом; запись — `OWNER_ANSWERS_DECISION_RECORD.md` §3.)*

Дополнительные услуги: **анимация / персонажи**, **дни рождения и праздники** — под тем же
зонтиком профессиональной труппы (это и есть отстройка от генерик-аниматоров).
