## Что мы строим (одна фраза)

Сайт **выездного детского театра** (бренд: **Miss Lana's Fairy-Tale Theatre**, зонтик **Miss Lana**, primary production domain `misslanatheatre.com` — см. `BRAND.md`) с базой в
Лос-Анджелесе: профессиональная труппа показывает **театрализованные шоу по добрым
сказкам** (живые костюмированные актёры) с **интерактивным перформансом для детей**;
плюс анимация, дни рождения, детские праздники. **Своего помещения нет** — театр
приезжает к зрителю. **Это НЕ «кукольный театр».**

## Формат продукта (важно — уточнено владельцем)

**Театрализованное шоу:** костюмированный спектакль по сказке (**~30 мин**) **+
интерактивный перформанс с детьми** (**игры, танцы, мыльные пузыри**). Живые
актёры. Всего **около часа** (~30 мин спектакль + ~30 мин интерактив), дети
**2–10 лет**. *(Длительность — приблизительная, без минута-в-минуту обещаний;
подтв. владельцем 2026-06-26. Формат «куклы внутри» подтверждён владельцем —
переписать guardrails отдельным пассом; запись — `OWNER_ANSWERS_DECISION_RECORD.md` §3.)*

## Для кого / какую боль решает

- **Первичная аудитория — англоязычный массовый рынок США.** В LA: **частные детские
  сады и Монтессори-школы**, **семьи (дни рождения, праздники)**, **школьные ассамблеи**.
- **Желаемый сегмент:** **государственные школы** (владелец хочет туда, но не знает как
  → отдельная go-to-market стратегия).
- **Опциональный вторичный слой:** украинская (и шире восточноевропейская) община —
  вторично; **не русская** (см. §«Идентичность»).
- **Боль (словами владельца):** «стало меньше заказов… заказы упали с момента, как
  перестала давать рекламу в соцсетях и звонить по телефону». Рост держался на ручном
  обзвоне + рекламе; перестали — поток упал. **Новых заказчиков почти нет.** Сайт должен
  заменить ручной обзвон постоянным каналом: **заявки + витрина + локальное SEO**.
