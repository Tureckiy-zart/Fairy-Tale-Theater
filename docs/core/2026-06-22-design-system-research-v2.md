## 1. Executive summary (рекомендация с первого взгляда)

**Рекомендованное направление — «Lantern Light» (Фонарик/Свет).** [HIGH]

Тёплая сказочная палитра: **лесные зелёные** (любовь владельца к зелёному) + **золотисто-янтарное свечение** на тёплом «пергаментном» кремовом фоне. Шрифты — **Fraunces** (характерный, но профессиональный сказочно-книжный serif) для заголовков + **Nunito** (тёплый humanist sans со скруглёнными окончаниями) для текста. Сквозной **сигнатурный мотив — тёплый свет/фонарик** (имя Svitlana = «свет»): мягкое свечение, лёгкое «мерцание», фонарик/искра как маркеры и акцент на hover у CTA.

Почему именно это:
- **Опирается на бриф буквально:** тёплое + зелёное + мотив «света/свечения» — заданы владельцем. [evidence: бриф/BRAND.md]
- **Обслуживает обе аудитории сразу:** спокойно-профессионально и достойно для **директоров садов/школ (B2B)** и тепло-волшебно для **семей**. [judgment, HIGH]
- **Обходит оба анти-паттерна:** (1) кричащий американский стиль «больших открытых ртов» — у нас сдержанно и премиально; (2) холодный корпоративный/шаблонный вид — у нас очень тепло и характерно. [HIGH]
- **Отстраивается от конкурентов:** Bob Baker = винтаж + «красный занавес»; PollyBilly = мультяшно-милый Wix; Puppet Theater on Wheels = прямой, открыто помечает 🇺🇦. Мы — **зелёный + золотое свечение + современная сказочная книга**, нейтрально-универсально. [evidence, HIGH]
- **Подтверждено эталонными брендами:** Lovevery доказывает, что **зелёная палитра читается премиально и надёжно**; связка зелёный + тёплый свет/фонарик **не занята ни на одном из 7 проверенных сайтов** — это наш самый чистый дифференциатор. [evidence (live-fetch CC), HIGH]
- **Сознательно уходит от дефолта ИИ-дизайна** (крем + контрастный serif + терракота): ведём **зелёным + золотым свечением + фонариком-сигнатурой**, а не терракотой. [judgment, HIGH]

**Альтернативы (для выбора владельцем):**
- **«Storybook Garden»** [MED] — более иллюстративно/ботанически, мягче и игривее; максимум очарования, но риск «скатиться в детсадовость» и чуть меньше премиальности для директоров.
- **«Warm Curtain / Evening Glow»** [MED] — театральнее и премиальнее (мягкий «занавес», насыщеннее золото); сильнее для B2B-доверия, но чуть «взрослее» для аудитории 2–10.

**Ключевые токены (кратко):** primary `forest-600 #41702C` (белый текст 5.87:1), акцент-свечение `glow-400 #EAAE35` (только тёмный текст), текст `ink #2A2520` на `cream #FBF7EF` (14.2:1), янтарный текст `glow-700 #8C5D14`. Типографика — **Fraunces + Nunito**. Иконки — **Phosphor** (Regular + Duotone). Радиусы мягкие (кнопки — pill), тени тёплые и рассеянные, движение деликатное, **motion-safe-first**, с паузой для циклов и поддержкой `prefers-reduced-motion`.

## 2. Брендовые ограничения (recap)

Из `BRAND.md`, `02_POSITIONING_AND_TONE.md`, `PROJECT_BRIEF.md` и `current_truth` задачи (последнее — авторитетно):

1. **Персона-лид.** Зонтик/лицо бренда — **Miss Lana** (тёплая, приветливая хозяйка; можно как дружелюбный иллюстрированный персонаж). Голос — от первого лица.
2. **Идентичность.** Профессиональный **живой костюмированный** детский театр сказок. **НЕ «кукольный/puppet».** Принцип: **широко в услугах, узко в идентичности.**
3. **Тон.** Тёплый, добрый, сказочный, «добрые сказки, как раньше». **Ярко, но НЕ агрессивно.**
4. **Цвет.** Тёплая палитра; владелец **любит зелёный**; мотив **«света/тёплого свечения»** (Svitlana = «свет» → фонарик/свечение как тихая визуальная нить).
5. **Анти-паттерны (обязательно избегать):** (1) кричащий «большие открытые рты», мультяшно-орущий; (2) холодный корпоративный / шаблонный / generic.
6. **Наследие.** Украинская труппа — **только тихий бэкстори** на «О нас». **Никакого** славянского/русского/украинского визуального кодирования. Бренд читается нейтрально-универсально, по-английски.
7. **Аудитория.** Англоязычный массовый рынок США: частные сады/Монтессори, дни рождения и семьи, дети **2–10**. ЛПР — директора садов + родители. Дизайн должен быть **надёжным/профессиональным** для B2B и **волшебным/тёплым** для семей.
8. **Картинки.** Реальные тёплые фото/видео шоу — в центре; система их **обрамляет и возвышает**.
9. **Доступность + перфоманс.** WCAG AA и Core Web Vitals (от этого зависит локальное SEO).
10. **Trademark — в процессе.** Логотип/вордмарк **не финализировать** — только направление.

## 5. Дизайн-направления (1 основное + 2 альтернативы)

### 5.1 ОСНОВНОЕ (рекомендую) — «Lantern Light» · уверенность HIGH

**Настроение:** зачарованно, тепло, надёжно, по-книжному. Лесные зелёные + золотое свечение на тёплом пергаменте; Fraunces + Nunito; **сигнатура — нить тёплого света/фонарика**; фото-форвард с иллюстрированной теплотой.

**Сигнатурный элемент:** мотив **света/фонарика** (Svitlana = «свет»). Проявления: мягкое свечение за ключевыми блоками, фонарик/4-лучевая искра как маркеры списков и метка активного пункта меню, лёгкое «мерцание» декоративных искр, **янтарное свечение на hover у кнопки** (тень `glow`). Один запоминающийся приём — остальное спокойно.

**Каких анти-паттернов избегает:** (1) не кричит — никаких «больших ртов», сдержанно и премиально; (2) не холодный/шаблонный — глубоко тёплый и характерный. Плюс **сознательно уходит от дефолта ИИ** (крем + serif + терракота): ведём зелёным + золотым свечением + фонариком, а не терракотой и не «голым» редакционным serif.

**Баланс аудиторий:** спокойная зелень + дисциплина сетки/типографики = доверие для директоров; тёплое свечение + книжная иллюстрация + персонаж Miss Lana = волшебство для семей.

### 5.2 АЛЬТЕРНАТИВА A — «Storybook Garden» · MED

Иллюстративнее и ботаничнее: больше зелени и садово-лесных мотивов, мягкие пастельные оттенки в тёплой семье, более «крафтовая»/тёплая типографика — **Fraunces с осью SOFT повыше** для заголовков, либо **Bricolage Grotesque** (дружелюбный humanist-гротеск с лёгкими «рукодельными» неровностями) + Nunito для текста; иллюстрация-форвард. **Плюс:** максимум очарования и тепла. **Риск:** при перегибе «детсадовость» и чуть меньше премиальности для ЛПР-директоров. Хорошо, если владелец хочет максимум charm.

> ⚠️ **Поправка v2:** прежний вариант с **Baloo 2 / Quicksand** для заголовков снят — это «кричаще/игрушечный» регистр, ровно анти-паттерн №1 (подтверждено проходом CC). Тёплую «круглость» берём из Nunito или Bricolage Grotesque, не из Baloo/Quicksand.

### 5.3 АЛЬТЕРНАТИВА B — «Warm Curtain / Evening Glow» · MED

Театральнее и премиальнее: глубже тёплые тона, мягкий мотив **сценического занавеса**, насыщеннее контраст, золото заметнее, атмосфера «вечернего спектакля». Типографика — **Lora** (тёплый книжный serif, изящный курсив для голоса Miss Lana) + **Plus Jakarta Sans** (чистый профессиональный sans для текста/UI). **Плюс:** сильнее B2B-доверие/премиальность. **Риск:** чуть «взрослее»/формальнее для семей 2–10. Отстраивается от винтажно-красного Bob Baker через **зелёный + тёплое золото**, не красный.

> Палитра, типографика и токены ниже даны для **основного** направления; альтернативы используют ту же инженерию токенов с другими акцентами/иллюстрацией/гарнитурами (отличия отмечены).

## 7. Типографика и шкала (+ направление вордмарка)

### 7.1 Гарнитуры (основное направление)

- **Дисплей/заголовки — `Fraunces`** (variable, OFL). Тёплый сказочно-книжный serif с осями `opsz` (9–144) и `SOFT` (0–100): характерный, но профессиональный и читаемый; один файл (variable) — хорошо для перфоманса. На крупных размерах — высокий `opsz`, вес **500–600** и лёгкий подъём `SOFT` для сказочной теплоты; на подзаголовках `SOFT` ниже, чтобы оставаться «деловыми» для директоров. **Тепло берём осью SOFT, а не тяжёлым весом** (не 800/900). [HIGH] *Почему не дефолт:* сознательно НЕ Playfair/«редакционный serif»; и независимо выбран проходом CC — сильная кросс-валидация.
- **Текст — `Nunito`** (variable, OFL). Humanist sans со **скруглёнными окончаниями** — ощущается добрым/мягким (в духе «kind tales like the old days»), но со щедрой высотой строчных и проверенной экранной читаемостью для длинной копи (родители) и B2B-деталей (цены/политики). Греется в паре с Fraunces. Веса 400/600/700. [HIGH] *Решение v2:* заменяет Nunito Sans из v1 — выбрана теплота скруглённых окончаний.
- **Утилитарный/мета/лейблы** — тот же `Nunito`, sm/xs, 600, лёгкий трекинг для капс-эйбров.
- **Рукописный акцент** (одно короткое слово вроде подписи «Miss Lana» или тэглайн про «свет») — **`Caveat`** (variable, OFL), **≤1 фразы на экран**, никогда для body/B2B-блоков; либо как **иллюстрация/ассет** (легче для перфоманса). Caveat предпочтительнее Mali (Mali не variable, тайско-первичный). [judgment]

**Шрифты альтернатив:** A «Storybook Garden» — Fraunces (SOFT↑) **или `Bricolage Grotesque`** + `Nunito` *(Baloo 2/Quicksand сняты — анти-паттерн)*. B «Warm Curtain» — `Lora` (serif, изящный курсив) + `Plus Jakarta Sans` (нейтральнее-профессионально).

**Анти-паттерн (НЕ брать для заголовков этого бренда):** `Fredoka`, `Baloo 2`, `Quicksand` — кричаще/«игрушечный», тяжёлый «детсадовский» регистр. `Outfit` — холодный геометрический, Latin-only, без курсива → тянет к анти-паттерну №2. Тёплую круглость — из Nunito/Bricolage Grotesque; доверие — из Fraunces/Lora.

### 7.2 Шкала (root 16px = 1rem) — наша, Tailwind-совместимая

| Токен | px | rem | line-height | вес/роль |
|---|---:|---:|---:|---|
| `text-xs`   | 12 | 0.75  | 1.5  | 600 — лейблы, подписи |
| `text-sm`   | 14 | 0.875 | 1.5  | 400–600 — мета, helper |
| `text-base` | 16 | 1.0   | 1.6  | 400 — **минимум для body** |
| `text-lg`   | 18 | 1.125 | 1.6  | 400 — лид-абзац |
| `text-xl`   | 20 | 1.25  | 1.5  | 600 — мелкие тайтлы карточек |
| `text-2xl`  | 24 | 1.5   | 1.35 | 600 — h4 |
| `text-3xl`  | 30 | 1.875 | 1.3  | 600 — h3 *(Fraunces)* |

Правила (с уточнениями CC): длина строки ≤ ~66–72ch; **дисплей-веса 500–600** (теплота — осью SOFT, не 800/900); крупные заголовки — трекинг −0.01…−0.02em; body — трекинг 0; капс-лейблы +0.06em; **каждый заголовок — в `clamp()`** (моб. — мягче, ~1.2; десктоп — полнее), напр. hero — `clamp(2.5rem, 5vw, 3.75rem)`; интерлиньяж: Display/H1 1.05–1.1, H2–H3 1.15–1.3, body 1.6 (Nunito читается лучше с щедрым leading). Family-токены: `--font-display: "Fraunces"`, `--font-body: "Nunito"`.

### 7.3 Перфоманс шрифтов (CWV)

- **Self-host через `next/font`** (App Router): по одному variable-woff2 на семью (Fraunces `wght+opsz+SOFT`, Nunito `wght`) вместо 6–8 статичных начертаний; убирает рендер-блокирующий запрос к `fonts.googleapis.com`.
- **Сабсет latin + latin-ext**, выкинуть неиспользуемые оси (WONK) и наборы → меньше КБ, бережёт LCP/CLS.
- `font-display: swap`; **preload** двух семей; **metric-overrides** (`size-adjust`/`ascent-override` в `next/font`) под фолбэки, чтобы CLS ≈ 0.

### 7.4 Направление вордмарка (НЕ финал — trademark в процессе)

Только направление: лок-ап с доминантой **«Miss Lana»** (зонтик-персона), под ним дескриптор «Fairy-Tale Theater». Имя — Fraunces (SOFT), дружелюбно, со **знаком-сигнатурой света**: маленький фонарик/4-лучевая искра (например, как «точка» над буквой или рядом с именем). Палитра forest + glow; можно в связке с иллюстрированным персонажем Miss Lana с фонариком. **Нейтрально-универсально, без славянских мотивов.** Дать иллюстратору 2–3 эскиза, финал — после trademark-clearance. Урок Bob Baker: **логотип с характером** (не generic sans) — лёгкий, но сильный дифференциатор. [direction-only]

### 8.1 Spacing (база 4px, согласовано с Tailwind)

```text
space-0:0  space-1:4  space-2:8  space-3:12  space-4:16  space-5:20
space-6:24 space-8:32 space-10:40 space-12:48 space-16:64 space-20:80
space-24:96 space-32:128
```
Семантика: `gutter` (боковые поля) 16 → 24 на десктопе; контейнер контента ~1280px по центру; `section-py` = `clamp(4rem, 8vw, 8rem)` (ритм секций 64→128); `card-padding` 24 (моб.) / 32 (десктоп); `stack-gap` 16 базовый, 24 между группами; проза `max-width: 72ch`.

### 8.2 Radius (мягко, дружелюбно)

```text
radius-xs:6  radius-sm:10  radius-md:14  radius-lg:20  radius-xl:28
radius-2xl:36  radius-pill:9999
```
Рекомендации: **primary CTA — pill (9999)**; карточки — `radius-lg`/`xl`; медиа/hero — `radius-xl`/`2xl`; поля ввода — `radius-md`; чипы/аватары — pill.

### 8.3 Elevation (мягкие, ТЁПЛО-тонированные тени)

Тон тени — тёплое какао `rgba(42,37,32, …)`, **не чистый чёрный** (это читается дёшево/шаблонно).

```text

shadow-lg: 0 14px 32px rgba(42,37,32,0.12)
shadow-xl: 0 28px 56px rgba(42,37,32,0.14)
/* «glow» — сигнатура света: на hover/active у CTA и световых акцентов */
shadow-glow: 0 6px 20px rgba(234,174,53,0.35)   /* янтарное свечение, по минимуму */
```

## 9. Иконография и иллюстрация (вкл. направление персонажа Miss Lana)

### 9.1 UI-иконки — `Phosphor` (решение v2; одна библиотека)

`Phosphor` (`phosphor-react`, MIT, ~1248 иконок, 6 начертаний, скруглённые концы — самый тёплый/округлый набор). **Одна библиотека на весь сайт, не смешивать.** [HIGH]
- **`Regular`** — для UI/навигации/функциональных иконок; штрих читается ровно, тон `ink`/`forest-700`; база **24px** (сетка 16/20/24/32/48).
- **`Duotone`** (тело ~30% + полный акцент) — для «искорки»/делайта: тело `forest`, акцент `glow` (или наоборот), на ≥24px. Это и есть приятная брендовая нить «света».
- **Макс. 2 начертания на экран** (обычно Regular + точечный Duotone), чтобы не пестрить.
- *Решение v2:* заменяет Lucide из v1; ради этого добавляем зависимость `phosphor-react`. (Если позже понадобится сверх-тонкий «chrome» — оставаться в Phosphor `Thin/Light`, а не подмешивать второй набор.)

### 9.2 Иллюстрация — техника под бриф иллюстратору (проход CC)

**Стиль:** текстурные **гуашевые** заливки + мягкий тёплый **«line-and-wash»**; **коричневая линия (не чёрная)**, средней толщины с лёгким сужением ~1.5–2px; низкоконтрастное **зерно**; мягкие, иногда «приоткрытые» края; **один тональный сдвиг — свет от фонаря «из-за кадра»**. Премиально для B2B, волшебно для семей; уходит от «плоского вектора SaaS» и высоконасыщенного мультика. [HIGH]

**Motif-kit (одна стилевая библиотека) + гард-рейлы:** фонарик (hero + CTA), мягкий **glow-ореол**, скруглённая **4-лучевая звезда**, **театральный занавес + портал-арка** (буквальный «театр» И рамка для фото), добрые мягкие зверушки (лисичка/совёнок/ёжик) для линии **& Friends** — спокойные глаза, **закрытые рты**.
**Гард-рейлы (обязательно держать единым):** одна толщина линии · одно зерно · одна палитра (text-safe vs decorative-only) · **одно направление света** (из-за кадра) · единая референс-библиотека. Всё **редактируемо до trademark-clearance**.

### 9.3 Мотив света/фонарика как микро-знак

Маленький фонарик или 4-лучевая искра: маркеры списков, разделители секций, метка активного пункта меню, лоадер, иконка CTA. Это и есть сквозная **сигнатура**. [HIGH]

### 9.4 Персонаж Miss Lana (направление; модель — Sago Mini)

Тёплая приветливая хозяйка/рассказчица в стиле **современной детской книжной иллюстрации** (та же гуашь + линия, что и motif-kit). **Округлый силуэт, читаемый в размере favicon.** Мягкая **полуулыбка**, закрытый/слегка приоткрытый рот, добрые спокойные глаза — **никогда** широко-открытого «орущего» рта (модель доброго персонажа — **Sago Mini**). Возраст-нейтральный гардероб рассказчицы в **шалфейно-моховой зелени**, опц. шаль со звёздами. **Сигнатурный реквизит — маленький тёплый фонарик с тихим ореолом** («свечение = имя = свет»). **Ноль славянского кодирования** (без матрёшек/фолк-орнамента/кокошника/кириллицы/«луковичных» куполов). Голос — от первого лица («I'll bring the story to you»). Использование: приветствие в hero, интро секций, пустые/успешные состояния, линия «& Friends», опц. небольшой «проводник». **Сосуществование с фото:** она — тёплый рассказчик, **реальные фото — доказательство**; на странице «О нас» — в паре с реальной труппой. **Только направление**; финал — после trademark-clearance (стайл-борд + 2–3 эскиза иллюстратору). ## 10. Работа с изображениями (real photos/video в центре)

- **Иерархия:** реальные фото/видео занимают **самые крупные и контрастные слоты**; иллюстрация **никогда не перекрывает лицо**. Иллюстрация — мягкий портал/скруглённые рамки, разделители (гирлянда занавеса, нити фонариков, россыпь звёзд), уголки, делайт у CTA, выглядывающая зверушка. **Сначала компонуем иллюстрацию, потом ставим фото; иллюстрацию держим легче по весу. Один регистр на экран.** (Эталон тёплого мягко-светового кадрирования — **Tonies**; ловушка — **KiwiCo**, где иллюстрация вытесняет фото.)
- **Грейд:** тёплый баланс белого, мягкий подъём теней, естественная насыщенность (без HDR/перешарпа), **единый по всей галерее**; грейдить под **тот же тёплый «свет фонаря»**. Дать лёгкий пресет/LUT-направление.
- **Кадрирование:** `radius-lg`/`xl`, `object-cover`, единые соотношения — hero 16:9 или 3:2; карточки 3:2 или 4:3; портреты 4:5; галерея — опц. masonry. Мягкая тень (`shadow-sm`/`md`).
- **Текст поверх фото:** тёплый градиент-скрим снизу (`forest-900`→прозрачный) до контраста **≥4.5** для текста (проверить при сборке) — **или** класть текст на сплошную карточку рядом с фото (безопаснее по доступности).
- **Видео:** muted, lazy, тёплый постер-кадр; **субтитры**; кнопка play — брендовая (фонарик/треугольник); **без autoplay со звуком**; пауза обязательна.
- **Alt-тексты:** описывать сцену + эмоцию; декоративная иллюстрация — пустой `alt`.

## 11. Движение / анимация (вкл. reduced-motion и перфоманс)

Деликатные, волшебные микровзаимодействия: мягкие fade, лёгкое «парение/мерцание» для мотива света, hover/focus, scroll-reveal. **Архитектура — motion-safe-first** (проход CC): по умолчанию всё статично, движение включается только там, где пользователь его не отключил.

### 11.1 Токены движения (реконсилиация v1 + Material 3 от CC)

```text
/* длительности — семантические */
--dur-instant: 100ms   /* press/active */
--dur-fast:    200ms   /* hover/focus/цвет */
--dur-base:    300ms   /* большинство переходов, модалки, кросс-фейд секций */
--dur-slow:    450ms   /* появления/scroll-reveal */
--dur-ambient: 600ms   /* мягкие декоративные */
/* амбиентные циклы (НЕ в UI-шкале): float 5–7s, twinkle 3–5s */

/* easing — именованные (значения Material 3) */
--ease-standard:      cubic-bezier(0.2, 0, 0, 1)      /* дефолт UI */
--ease-decelerate:    cubic-bezier(0.05, 0.7, 0.1, 1) /* появления/вход — мягко «доезжает» */
--ease-accelerate:    cubic-bezier(0.3, 0, 1, 1)      /* уходы/выход */
--ease-gentle-spring: cubic-bezier(0.34, 1.2, 0.64, 1)/* мягкий «settle» (hover, бобинг фонарика) — овершут СИЛЬНО прибран, не «мультяшная пружина» */
```

### 11.2 Паттерны (свойство → токен)

- **Hover-lift** (карточки/CTA): `translateY(-2…-4px)` + тень `md→lg`, `--dur-fast` + `--ease-gentle-spring`.
- **CTA hover:** фон `forest-600 → 700`, лёгкий `scale(1.02)`, появляется `shadow-glow` (янтарное свечение), `--dur-fast`.
- **Press/active:** `scale(0.97)`, `--dur-instant`.
- **Link/text hover:** только цвет + opacity, `--dur-fast`.
- **Focus-visible:** кольцо **мгновенно** (без задержки) — доступность.
- **Scroll-reveal (ниже сгиба):** `opacity 0→1` + `translateY(16px → 0)`, `--dur-slow` + `--ease-decelerate`, **one-shot** (unobserve после); сдвиг малый (12–24px); stagger детей 60–90ms с потолком ≤~400ms на группу; **резервировать место** (без CLS). Триггер чуть заранее (`rootMargin: '0px 0px -10% 0px'`, threshold ~0.15).
- **Модалка/лайтбокс фото:** `opacity` + `scale(0.98→1)`, `--dur-base` + `--ease-decelerate`.
- **Сигнатурное «мерцание/свечение»:** фонарик мягко «дышит» `translateY(0→-6px→0)` 5–7s; искры — `opacity 0.6→1→0.6` + `scale 1→1.06`, 3–5s, **со сдвигом, НИКОГДА не синхронно**. Максимум **2–3** амбиентных элемента в движении одновременно; остальное — статично. **Только декор.**

### 11.3 `prefers-reduced-motion` (motion-safe-first)

- **По умолчанию движение выключено**; включаем только внутри `@media (prefers-reduced-motion: no-preference)` (Tailwind `motion-safe:`). Так старые браузеры и reduced-motion-юзеры **автоматически** получают статичную, полностью рабочую страницу.
- Под `reduce`: убрать многоосевое/крупное движение, parallax, scale, spin и **циклы** (float/twinkle/пульс свечения). **Оставить** мягкий opacity-кросс-фейд/цвет, чтобы смена состояния всё ещё считывалась (где движение несло смысл — дать без-движущуюся альтернативу).
- **Пауза для циклов (WCAG 2.2.2):** любая зацикленная/амбиентная анимация — с паузой/стопом; держать перцептивную длительность цикла **< 5s**.
- **Никогда** не передавать состояние только движением.
- Бэкстоп (на всякий случай, не основная стратегия): глобальный `@media (prefers-reduced-motion: reduce){ *,*::before,*::after{ animation-duration:.01ms!important; animation-iteration-count:1!important; transition-duration:.01ms!important; scroll-behavior:auto!important } }`.

### 11.4 Перфоманс / Core Web Vitals

- Анимировать **только `transform` / `opacity`** (и `filter` точечно для свечения) — композитор; **никогда** layout-свойства (`width/height/top/left/margin/padding/font-size`) и **никогда `transition: all`** (тихо анимирует layout → топ-источник CLS).
- **Не анимировать/не задерживать LCP-элемент** (hero-картинка/видео/H1 рендерятся в финальной позиции сразу; reveal — только ниже сгиба). Амбиент над сгибом — стартует **после** LCP.
- Реализация reveal: нативная **CSS `animation-timeline: view()`** за `@supports (animation-timeline: view())` (off-main-thread) со статичным фолбэком; **или** лёгкий **IntersectionObserver** (добавляет `.is-visible` один раз → CSS делает анимацию). **Без** scroll-listener'ов; **без** тяжёлых либ (GSAP/AOS/Lottie/большой Framer-Motion) ради простых fade.
- `will-change: transform/opacity` — только на активно-анимируемых элементах, снимать после.
- lazy-load медиа/видео ниже сгиба; задавать `width/height/aspect-ratio` (без CLS). Проверять CLS ≈ 0 в Lighthouse/PageSpeed и что движение полностью замирает при OS «reduce motion».

- **Не** класть белый текст на золото (`glow-*`); **не** брать `glow-600` для body-текста (только `glow-700`); **не** использовать `ink-muted` для мелкого body.
- **Не** анимировать LCP/над-сгибом; **не** использовать `transition: all` и layout-свойства; **не** оставлять цикл без паузы (WCAG 2.2.2); амбиент — макс. 2–3 элемента, не синхронно.

- **Не** дрейфовать к дефолту ИИ (крем + serif + терракота) — отстройка = зелёный + свечение + фонарик-сигнатура.
